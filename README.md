# shinyADI

<!-- badges: start -->

<!-- badges: end -->

The goal of shinyADI is to provide an intuitive, interactive interface
to the `get_adi()` function in the
[sociome](https://github.com/ClevelandClinicQHS/sociome) R package.

Use the shiny app at <https://nikkrieger.shinyapps.io/shinyadi/>.

Or, run it using:

``` r
shiny::runGitHub("shinyADI", "NikKrieger")
```
